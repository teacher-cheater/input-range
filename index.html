Тестовое задание

Реализовать форму на javascript без использования готовых библиотек с единственным полем принимающим значение диапазона чисел, поле представляет собой два спаренных инпута с виджетом в виде ползунка с двумя контроллерами для управления, позиции контроллеров и значений инпутов должны быть синхронизированы

При сабмите формы на бекенд отправляются числовые значения инпутов, так же у формы должна быть возможность установить значения при инициализации и возможность установить новые значения после инициализации, реализовать можно на кнопках обрабатывая событие “клик”
